# راهنمای تبدیل به فایل اجرایی ویندوز (.exe)

## چرا من نمی‌توانم exe را برایتان بسازم؟
ابزار ساخت exe (PyInstaller) فقط برای همان سیستم‌عاملی که رویش اجرا می‌شود خروجی می‌سازد؛
یعنی از لینوکس نمی‌شود exe ویندوزی ساخت (Cross-compile نمی‌کند). محیط اجرای من لینوکس است،
پس باید این مرحله را خودتان روی یک کامپیوتر ویندوزی انجام دهید — اما فقط ۳ دستور ساده است.

## پیش‌نیاز: نصب Python روی ویندوز
1. به سایت python.org بروید → Downloads → آخرین نسخه پایتون ۳ را دانلود کنید.
2. هنگام نصب، حتماً تیک **"Add python.exe to PATH"** را بزنید.
3. برای اطمینان، Command Prompt (cmd) را باز کنید و بزنید:
   ```
   python --version
   ```

## مرحله ۱: استخراج پروژه
فایل `accounting_core_phase1.zip` را در یک پوشه (مثلاً `C:\Hesabdari`) استخراج کنید.

## مرحله ۲: نصب PyInstaller
در Command Prompt:
```
cd C:\Hesabdari\accounting_core
pip install pyinstaller
```

## مرحله ۳: ساخت فایل exe

### نسخه گرافیکی (پیشنهادی) — پنجره، دکمه، جدول
```
python -m PyInstaller --onefile --noconsole --name HesabdariHoshmand gui.py
```
- `--noconsole` یعنی پنجره سیاه ترمینال دیده نشود؛ فقط پنجره گرافیکی باز می‌شود.

### نسخه متنی (منوی کنسول)
```
python -m PyInstaller --onefile --console --name HesabdariHoshmandConsole main.py
```
- `--onefile` یعنی همه‌چیز در یک فایل exe واحد بسته‌بندی شود.

می‌توانید هر دو را بسازید و هرکدام را ترجیح دادید استفاده کنید.

بعد از چند ثانیه، فایل نهایی اینجا ساخته می‌شود:
```
C:\Hesabdari\accounting_core\dist\HesabdariHoshmand.exe
```

## مرحله ۴: اجرا
همان فایل `HesabdariHoshmand.exe` داخل پوشه `dist` را دوبار کلیک کنید.
- در همان پوشه‌ای که exe قرار دارد، یک فایل `hesabdari.db` به‌صورت خودکار ساخته می‌شود —
  این پایگاه داده شماست؛ اگر می‌خواهید داده تازه شروع شود، همین فایل را حذف کنید.
- برای توزیع به کامپیوترهای دیگر: کافی است فقط همان یک فایل `HesabdariHoshmand.exe` را
  کپی کنید (به نصب پایتون روی کامپیوتر مقصد نیازی نیست).

## اگر خطا دیدید
- **`'pip' is not recognized`** → پایتون درست نصب/به PATH اضافه نشده؛ نصب را تکرار کنید و تیک PATH را بزنید.
- **آنتی‌ویروس فایل exe را مسدود کرد** → طبیعی است (فایل‌های PyInstaller گاهی به‌اشتباه هشدار می‌دهند)؛
  گزینه "Allow" یا "More info → Run anyway" را بزنید یا از آنتی‌ویروس استثنا بگیرید.
- **پنجره سریع باز و بسته می‌شود** → از داخل Command Prompt اجرا کنید (نه دابل‌کلیک) تا خطا را ببینید:
  ```
  cd C:\Hesabdari\accounting_core\dist
  HesabdariHoshmand.exe
  ```

## قبل از ساخت exe: تست مستقیم رابط گرافیکی
می‌توانید بدون ساخت exe هم گرافیکی را امتحان کنید (چون tkinter با پایتون ویندوز از قبل نصب است):
```
cd C:\Hesabdari\accounting_core
python gui.py
```
باید یک پنجره با تب‌های «کدینگ حساب‌ها»، «ثبت سند حسابداری»، «تراز آزمایشی» و «رهگیری عملیات» باز شود.

## قدم بعدی پیشنهادی
اگر ماژول‌های بیشتری (انبار، خزانه‌داری، حقوق‌ودستمزد) یا گزارش خروجی PDF/Excel بخواهید،
فقط بگویید کدام را اول می‌خواهید تا روی همین رابط گرافیکی اضافه کنیم.
